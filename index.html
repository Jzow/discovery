Discovery

通过Rust了解单片机的世界!

这本书目前有两个版本。它们都提供了单片机介绍以及如何使用Rust。第一个版本较旧，使用F3发现电路板，而第二种是较新的，使用micro:bit电路板。

  • 使用micro:bit，阅读较新版本的书
  • 使用F3电路板阅读旧版本的书
  • 开始处理此存储库中的示例
  • 你有问题吗？
    • 看看我们的discussions section on GitHub
    • 也许它已经被回答了
    • 如果没有，开始新的讨论
  • 你发现了问题？
    • 看看我们的issues on GitHub
    • 也许已经有了一个解决办法
    • 如果没有，请打开一个新的或者更好pull request来解决它
  • 玩得开心!
